Who is Planbot?

Planbot is a chatbot designed to provide users with concise planning information as well as important legislation and market research.

What can it do?

Definitions

Ask Planbot for a definition to over 750 property and planning terms.

Information

Planbot can inform you of different types of use classes and permitted development.

Documents

Planbot is a powerful tool for finding planning policy and legislation.

More platforms and features are planned for the future. See our white paper for more information.

Slack integration?

Slack commands allow you to quickly request data from Planbot whilst working on a project:

/define Get a definition
/use Find use class information
/project Find permitted development information
/doc Request legislation and policy documents
/lp Request a local plan

What about an API?

Anyone can freely access our API and, as a taster, you can see how it works below. Visit our API page for more details.

GET https://api.planbot.co/lp/

Can I help?

Yes! Planbot is an open source project. That means the source code is freely available to modify and distribute in accordance with the GNU General Public Licence version 3.

We actively encourage collaboration from all backgrounds, no matter how experienced you are. If you would like to contribute in any capacity, whether it be to the logo or the source, get in touch.

All feedback is welcome. As the project is work in progress, every positive and negative comment you make can help to shape the future of Planbot.
